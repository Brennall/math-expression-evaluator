INTRODUCTION

A simple to use and efficient evaluator for Math expression using Javascript

How to use

Basic usage

Parse an expression using var parsed = a.parse(exp); which returns a Array which will further processed by toPostFix and postFixEval That Object is needed to be converted to postfix var postfix = parsed.toPostfix(); which converts the array to Postfix notation and return new one Now to get the value of expression use postfixEval var result = postfix.postfixEval(); which is results

Using variables

If your expression has user defined variables like hundred = 100 then parse takes another argument for user defined variable var parsed = a.parse(exp,{hundred:100}); To evaluate variable also pass the key-value pair to postfixEval var result = postfix.postfixEval({hundred:100});
